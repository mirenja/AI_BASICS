## Running the Projects

### 1. Knights Puzzle

1. Navigate to the knights folder:

``` bash
    cd knights
```

2. Run the puzzle solver:

``` bash
python3 puzzle.py
```

The program will solve the knight and knave puzzles and print the solution.

### 2. Minesweeper AI

1. Navigate to the minesweeper folder:

``` bash
cd ..minesweeper
```

2. Install the required Python packages:

``` bash
pip install -r requirements.txt
```

3. Run the Minesweeper UI:

``` bash
python3 runner.py
``` 

Play the game using the UI. 
To let the AI make a move, select the AI Move option, and the AI will automatically select a cell.
