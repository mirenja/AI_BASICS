<h1 align=center><strong>Knights and Knaves Homework</strong></h1>

This exercise lets you practice writing formulas in propositional logic. 

# Credit:
This exercise and a lot of the code have been taken from the course CS50: Introduction to Computer Science (https://pll.harvard.edu/course/cs50-introduction-computer-science) 


# Setup
This project does not require additional dependencies beyond what's included in a standard Python installation.

# How to run?
You start the program by starting puzzle.py
The script does not require additional arguments.

# Your Task

Your task is to model riddles 1, 2 and 3 in puzzle.py.
The best way to approach this is to read through the file, make sure you understand the documentation and the write the riddles.

Once the riddle is implemented you can execute puzzle.py to test your solution.
